Future expansion

Future outpatient specialties for MDS.

MDS is designed so the same structured platform logic can eventually extend into additional outpatient specialties where repeatable documentation patterns matter.

This future expansion is intentionally outpatient-only. It is designed for clinic-facing specialties rather than inpatient, operative, or hospital-wide note systems.

How expansion fits

One platform, many specialties

  • Extend the same structured system mindset into additional outpatient specialties.
  • Use specialty expertise to define families, workflows, and high-value documentation patterns.
  • Preserve one premium platform identity while allowing specialty-specific depth and pricing.

Potential specialty directions

These are examples of outpatient specialties where MDS could eventually extend if the platform continues to mature.

Pediatrics

Well-child visits, acute clinic care, developmental follow-up, and counseling-heavy outpatient patterns.

Sports medicine

Injury follow-up, procedure support, rehab counseling, return-to-play workflows, and structured reassessment language.

Ophthalmology clinic

Outpatient clinic documentation systems and follow-up patterns distinct from OR documentation.

Outpatient OB/GYN

Clinic-side preventive, follow-up, and counseling workflows while explicitly excluding operative portions.

Important scope rule

This page is intentionally framed around outpatient documentation systems only.

Any future specialty expansion is for clinic-based documentation architecture. It is not currently intended to cover inpatient note sets, operative documentation, or hospital-wide workflow products.
